# Azure IaC Implementation using Terraform

## Overview
This repository contains Terraform code to deploy a multi-tier architecture on Azure.

## Structure
- terraform/main.tf
- terraform/backend.tf
- terraform/modules/network
- terraform/modules/app
- terraform/modules/db
- azure-pipelines.yml

## Commands
```bash
terraform init
terraform plan
terraform apply
```
